
# EmbodMoCap

A unified pipeline for processing embodied motion capture data using multi-stage processing.

## Documentation

For detailed documentation and workflow instructions:

**[English Documentation](../docs/embod_mocap.md)**

**[中文文档](../docs/embod_mocap_zh.md)**

## Key Features

- **Low-cost setup**: Only requires 2 iPhones (Pro) for human-scene motion capture
- **No wearable devices**: Completely marker-free and sensor-free motion capture
- **Indoor/Outdoor scene**: Works in arbitrary environments without camera calibration setup
- **Downstream Research**: Dataset export for evaluation and downstream tasks
